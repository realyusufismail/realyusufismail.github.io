  2. How to add Crafting Recipes.

Important

Use a Data generator if you are adding more than a couple of items in your mod. Part 1 will be creating a recipe in a json file were as part 2 is adding a recipe in a Data generator. So lets get started.

First of all

You want to go src/main/resources/data/modidfolder. for example mine is bentenmobmod which is my modid.

After that you want to create a recipes folder in which you will create a json file for the recipe.

Next you will create a json file called your item name or armour name for example i will use a sword in my mod and i will call the file heatblast_sword.json

In the json file we will start by adding this code

''' { "type": "minecraft:crafting_shaped", '''

This is the type of crafting you want to use. This is for the crafting table if you want to do smelting for example it would be smelting instead of crafting_shaped

You are done with this tutorial.
