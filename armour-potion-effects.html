How to add potion affects to armour

Step 1: create a folder called events as seen here.

Step 2: make a file in the event folder called yourarmoursetnameArmor.

Step 3: after that at this point of your code you will insert this 

 extends ArmorItem implements IForgeItem
This is need for the event.

Step 4: Insert this under your code. This is important for the event.

 
public ExampleArmor(IArmorMaterial materialIn, EquipmentSlotType slot, Properties builder) {	

super(materialIn, slot, builder);	
														
}

Step 5: insert this code. This is an @Override method. Every time there is armor tick the effects load.

 
@Override
public void onArmorTick(ItemStack stack, World world, PlayerEntity player) {

Step 6: Now specify the items. were the legs go for example. This is need for the next part of the code. Now we will add the potion affects. The game also need this like when head,chest,legs and boots give affect. thats why you alo have to say which is which for the next part. in step 7 you will do this boots.getItem() == ItemInit.HEATBLAST_BOOT.get() which mean that the boots is HEATBLAST_BOOT which means the game knows that if HEATBLAST_BOOT is equiped it is the boots.

	ItemStack boots = player.getItemBySlot(EquipmentSlotType.FEET);
		ItemStack legs = player.getItemBySlot(EquipmentSlotType.LEGS);
		ItemStack chest = player.getItemBySlot(EquipmentSlotType.CHEST);
		ItemStack head = player.getItemBySlot(EquipmentSlotType.HEAD);

Step 7: Now we say if legs equal example leggings chest equal example chestplate ect. This is important if we want the effect to applied when all armour is worn and when a piece of that armor is taken off the effects are removed. After that we put what potion effects we want. like if we want fire resistance when armour worn. 100 in the code is 10 seconds in agme which mean after the one peace of the armour is taken of the potion only remains for 10 seconds and the 1 is the level of the potion affect. like FIRE_RESISTANCE 1.

if (boots.getItem() == ItemInit.HEATBLAST_BOOT.get() && legs.getItem() == ItemInit.HEATBLAST_LEGGINGS.get() && chest.getItem() == ItemInit.HEATBLAST_CHESTPLATE.get() && head.getItem() == ItemInit.HEATBLAST_HELMET.get()) { 
	player.addEffect(new EffectInstance(Effects.FIRE_RESISTANCE, 100, 1, false, false, true));
}
note change heatblast to your armor
